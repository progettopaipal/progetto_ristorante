# progetto_ristorante
## Set up the project
- scaricare il file zip
- estrarlo
- eseguire il file Database.sql con mysql
- recarsi nella cartella con il terminale e digiatre sudo node server.js
- Alla fine collegarsi a localhost:4000/index.html
## Tecnologie usate
- html,css,javascript
- nodejs
- Bootstrap
- Eclipse web
- Mysql<br>
